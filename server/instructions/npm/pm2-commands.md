# Pm2 Commands

```bash
  # navigate to your repo server folder

  # start
  pm2 start ecosystem.config.js
  pm2 start

  # status
  pm2 status

  # monitor
  pm2 monit

  # Save current processes
  pm2 save

  # stop
  pm2 stop <name>
  pm2 stop all
  
  # delete
  pm2 delete <name>
  pm2 delete all
```
