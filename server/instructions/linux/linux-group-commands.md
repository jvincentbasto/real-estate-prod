# Linux User Commands

## Commands

- **User commands**

```bash
  # check groups
  cat /etc/group

  # check group
  groups <groupname>
  id <groupname>

  # add group
  sudo groupadd <groupname>

  # rename group
  sudo groupmod -n <newgroupname> <oldgroupname>

  # delete group
  sudo groupdel <groupname>
```
